# ⚖️ CodeHub

این یک پروژه شخصی برای جمع‌آوری کامل‌ترین، دقیق‌ترین و منقح‌ترین متن قوانین برای موارد استفاده احتمالی در آینده است. هر کسی می تواند استفاده کرده یا به بهبود این  پروژه کمک کند.

[English](README.md) | [عثمانلیجه](README.ota.md) | [Türkçe](README.tr.md)

## ویژگی‌ها

- **متن کامل قوانین**: شامل مجموعه‌ای از کدها و مقررات قانونی که شامل، اما نه محدود به:
	- قانون مدنی عثمانی، معروف به مجله (۱۸۶۸-۱۸۷۶)
	- قانون مدنی ایران (۱۹۲۸-۱۹۳۶)
	- قانون اساسی مشروطه (۱۹۰۶)
- **حرکه‌گذاری حروف**: نشانه‌گذاری صحیح حروف صدادار برای خواندن آسان‌تر و درک بهتر.
- **عاری از اشتباهات تایپی**: سعی کردم اشتباهات تایپی را به حداقل برسانم.
- **دسترس در فرمت‌های مختلف**: مانند .md و .epub

## نحوه استفاده

کدها به صورت فایل‌های Markdown و EPUB برای دانلود در بخش انتشارها در دسترس هستند.
